# DuolingoSimulator

A premium, high-fidelity simulation of the language learning experience.

## Tech Stack
- HTML5
- CSS3 (Modern Vanilla with Glassmorphism)
- Vanilla JavaScript

## Getting Started
Just open `index.html` in your browser to start your language learning journey.

## Features
- ✨ Stunning Modern UI
- 🔥 Streak Tracking (Simulated)
- 🎓 Interactive Lesson Mockups
- 📱 Fully Responsive Design
